> Bacteria filha da puta microbio do caralho

## Compatibilidade

- Mac
- Linux
- Windows

## Rodando com [npx](https://www.npmjs.com/package/npx)
Rode o comando:

    npx microbiodocaralho


## Rodando via CLI
Instale globalment e rode o comando:

    npm install -g microbiodocaralho

    microbiodocaralho
